# breakout-game
Simple Breakout Game using ACM library as a part of Standford CS course 


**Author:** Artem Tiupin

**Composer:** Marichka Kvertie


### Launching the game:

To run the game, please add two .jar files to your project:

- cs-java-lib-GRAPHICS

- cs-java-lib-TEXT
